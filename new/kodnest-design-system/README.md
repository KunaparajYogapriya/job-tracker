# KodNest Premium Build System

A premium SaaS design system for serious B2C product companies.

**Philosophy:** Calm, Intentional, Coherent, Confident.

---

## Quick Start

```html
<link rel="stylesheet" href="tokens.css">
<link rel="stylesheet" href="base.css">
```

---

## Files

| File | Purpose |
|------|---------|
| `tokens.css` | Design tokens (colors, spacing, typography) |
| `base.css` | Base styles, components, layout |
| `SPECIFICATION.md` | Full design system specification |

---

## Design Tokens Summary

- **Colors:** #F7F6F3, #111111, #8B0000, #4A6B4A (4 max; warning uses accent)
- **Spacing:** 8, 16, 24, 40, 64px
- **Layout:** 70% workspace / 30% panel

See `SPECIFICATION.md` for complete documentation.
